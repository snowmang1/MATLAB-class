	\section*{Question}
		\indent Please look at the nine set theory axioms and examine them personally.
		That is, express what is difficult about understanding them;
			what is easy about understanding them;
			how they describe sets.
		Dive into the axioms with as much detail as you have interest.

	\section*{Assignment}
		\indent the turn-in work for this assignment is a paper.
		If you want a format Id say AMS or IEEE but the format does not matter much to me.
		The paper has no minimum or maximum, simply write a paper you could be proud of.
		Please turn in a pdf file to the canvas.

	\section*{Tips}
		\indent recall the point of this kind of problem is not necessarily a \textit{correct} answer,
			rather an exploration.
		It will benefit you to find something about this that interests you and focus in on that.
